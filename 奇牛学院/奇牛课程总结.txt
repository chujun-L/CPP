一、C/C++基础
    1、用于计算的基本数据类型(unsigned signed)
        short、int、long、long long、float、double、long double
    
    Q：它们占多少个字节
    A：在32位及64位的系统中会有不同。而short是确定的，它占两个字节；在32位系统中，int long都占4字节，在64位系统中，int占4字节，long占8字节

    Q：精度？
    A：float：   1bit（符号位） 8bits（指数位） 23bits（尾数位）            占4个字节
       double：  1bit（符号位） 11bits（指数位） 52bits（尾数位）           占8个字节
       它俩的数值范围由指数位确定，精度由尾数位确定
        (像8.24默认为double类型。8.24F(float)，8.24L(long double))

    Q：printf宽度控制和精度控制
    A: [输出最小宽度][.精度][长度][类型]

    2、字符及字符串
        char    它占1个字节     ascii码：十进制(65~90: A~Z), (97~122: a~z)，大写与小写相差32。(48~57: 0~9)
        字符串   字符串是字符的集合。在C语言中，可以用字符数组char str[]或字符指针char *p来表示；在C++中，可以用C风格的字符串string及string类表示；字符串以空字符'\n'结束；

    3、布尔类型     true    false

    4、运算符   逻辑运算符&& || !   条件运算符> < == >= <=  位运算& | 

    5、数组
        同一种数据类型的集合，首地址可以用&arry[0]或arry表示。多维数组，例如二维数组，可以表示地图的x,y坐标；数组的应用时要注意不要越界访问
    
    6、指针
        它存储的是一个有效的内存地址，通过解引用(*)获取地址的内容。指针的运算(+, -)，与数组及字符串的下标对应，方便在连续的地址空间寻址；相减的用法比较特殊，可以求地址的偏移，例如内核中的宏offsetof及container_of是典型的应用例子。

    7、引用
        它是已经初始化的变量的别名，实质上也是指针(type *const p)。

    8、关键字
        static      它可以修饰变量、函数、类的函数成员、类的数据成员。修饰变量时，例如函数中的变量被修饰为static时，它的存储空间不在栈上，而是在全局初始化的data段，但它的作用域还是在该函数内部，函数被释放时，它不会被释放掉；修饰函数时，该函数的作用域只能是该文件内，其它文件不能通过extern来访问；修饰类的函数成员时；修饰类的数据成员时，表示该类的数据成员不属于对象，并且多个对象共用一份该数据成员。

        const       限定修饰符，即被该关键字修饰的对象不可被修改，例如变量，被修饰后变常量；类方法后面的const表示不能修改该方法里面的变量；
        inline      修饰函数，表示编译时将该函数直接插入被调用处，从而避免函数调用时栈的开销，适用过经常被调用的小函数。
        extern      当该文件中需要引用另外的文件的变量时使用。

        typedef     给一个比较长的数据类型起个简短的别名，例如结构体，指针函数
        define      它是一个预处理指令，通常用来定义一个常量，替换程序中用到的常量。当表示有数据类型意义的时候，不要与typedef混淆。例如：                   #define POINT (int *)   POINT a, b;
